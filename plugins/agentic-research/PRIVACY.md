# Privacy Policy

`agentic-research` operates on files inside user-selected workspace and on
tools available in host agent environment.

## What plugin accesses

- research project files in workspace
- notes, drafts, proofs, experiment outputs, and bibliography files
- user-provided prompts and local plugin assets

## What plugin does not do by itself

- it does not run a hosted backend owned by plugin author
- it does not collect analytics by default
- it does not transmit data anywhere except through tools and models chosen by
  host environment and user

## Responsibility split

Network access, model providers, telemetry, and authentication are governed by
host application and any external tools the user enables.
